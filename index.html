Requisitos Funcioais

  • O sistema deve incentivar a leitura
  • O sistema deve ter um lembrete automático para devoluções
  • O sistema deve recomendar livros baseados no que ja foram lidos
  • O sistema deve oferecer um espaço para discurção sobre os livros
  • O sistema deve registra os livros lidos por mais tempo
  • O sistema deve oferecer um biblioteca digital integrada no sistema escolar
  • O sistema deve oferecer um lugar em que os alunos consigam visualizar quais livros estão disponíveis para reservar
  • O sistema deve permitir que os alunos registrem sugestões de livros
